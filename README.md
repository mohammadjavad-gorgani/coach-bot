# 🤖 یار مربی (Coach Assistant Bot)

یار مربی یک ربات تلگرام یا یک بازو در بله برای مدیریت باشگاه‌های ورزشی است که به مربیان کمک می‌کند:

- 🧍 ثبت و مدیریت هنرجویان
- 📆 مدیریت کلاس‌ها
- ✔️ حضور و غیاب کلاس‌ها
- 💳 ثبت پرداخت شهریه و خرید تجهیزات
- 📊 دریافت گزارش‌ها

---
